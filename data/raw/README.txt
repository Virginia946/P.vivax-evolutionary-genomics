This directory contains simulated summary statistics for five different models (E,F,G,H, I) of Plasmodium vivax
arriving in the Americas. Each tsv file contains ~20,000 simulations of each specific model. Next, a description of each summary statistic:

pi_col: Nucleotide diversity of the Colombian Plasmodium population
pi_ebro: Nucleotide diversity of the Spanish plasmodium population
Segregating_sites_col: Segregating sites in the Colombian population
Segregating_sites_ebro: Segregating sites in the Spanish population
Fst (Fixation Index): Value between Colombian and Spanish populations
Bin_*_col: Bins made with AFS for Colombian populations
Bin_*_ebro: Bins made with AFS for Spanish populations
